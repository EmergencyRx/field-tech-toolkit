# Field Tech Toolkit

Scripts and documentation to support Linux edge PCs, IP cameras, ALPR-style systems, NVRs, and roadside or NEMA enclosures.

## Goals and Features

- Camera validation: reachability, DNS, NTP, RTSP/HTTP checks
- Linux health checks: disk usage, SMART, temperature, services, logs
- Network checks: gateway, MTU, VLAN, PoE, link quality
- Site survey and installation templates
- Handover and sign-off documentation
